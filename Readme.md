# AstroLip

Astronaut intention recognition through lip movement analysis.

## Overview

AstroLip explores whether astronaut intentions can be recognized from visual lip movements alone, reducing reliance on audio communication in noisy or constrained environments.

## Goals

- Build a domain-specific lip movement dataset
- Recognize mission-related commands
- Evaluate visual speech as an astronaut communication modality

## Repository Layout

- `anaconda_projects/` - local experimentation and notebooks
- `dataset/` - raw data, annotations, and splits
- `docs/` - design notes, references, and project documentation
- `experiments/` - training runs, configs, and results
- `models/` - saved model artifacts and checkpoints
- `papers/` - papers, citations, and literature notes
- `results/` - figures, tables, and evaluation outputs
